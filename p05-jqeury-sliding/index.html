P05 jQuery Sliding - Klik Disini
With jQuery you can create a sliding effect on elements.
jQuery has the following slide methods:
slideDown()
slideUp()
slideToggle()

1. jQuery slideDown()Method
The jQuery slideDown() method in used to slide down an element
syntax: $(selector).slideDown(speed,callback);

2. jQuery slideUp() Method
he jQuery slideDown() method in used to slide up an element
syntax: $(selector).slideUp(speed,callback);

3. jQuery slideTogle() Method
The jQuery slideToggle() Method toggles between the slideDown() and slideUp() methods.
