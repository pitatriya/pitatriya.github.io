Selamat datang di halaman home website portofolio saya


Ini merupakan website portofolio. Website ini dibuat menggunakan text editor Sublime Text dengan bahasa didalamnya yaitu html, css dan Javascript. Menu disebelah kiri merupakan link menuju hasil challange Qwars di beberapa pertemuan minggu kemaren, yaitu membuat table, form pendaftaran, sliding dan Single Page Application. Lalu dibaian bawa ada card yang terdapat foto, nama, dan nim saya. Untuk menu yang ada disebelah kanan merupakan menu-menu utama di website ini.

Sekilas tentang Qwars
Qwars sendiri merupakan website untuk matakuliah Web Pemrograman Dasar dengan dosen Pengampu Pa Iin Solihin. Di website tersebut terdapat banyak menu yang dapat dikerjakan yaitu diantaranya membuat dan mengerjakan soal, mengerjakan challenge, melakukan presensi dan banyak fitur lainnya.
